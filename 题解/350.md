# 350. 两个数组的交集 II

解法一：Map
使用hashMap先把数组1放入，然后在遍历数组2从map中找是否有相同数据，有就加入返回值，并且map对应次数-1，
```
class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        int n1 = nums1.length;
        int n2 = nums2.length;
        Map<Integer,Integer> map = new HashMap();
        List<Integer> list = new ArrayList();

        for (int i = 0 ; i < n1 ; i++) {
            Integer count = map.get(nums1[i]);
            count = count == null ? 1 : ++count;
            map.put(nums1[i],count);
        }
        
        for (int i = 0 ; i < n2 ; i++) {
            Integer count = map.get(nums2[i]);
            //次数count不为负数
            if(count != null && count != 0){
                list.add(nums2[i]);
                map.put(nums2[i],count-1);
            }
        }
        
        int[] ans = new int[list.size()];
        for (int i = 0 ; i < list.size();i++){
            ans[i] = list.get(i);
        }
        return ans;

    }
}
```


解法二：双指针

先对数组进行排序后，然后用双指针分别指向两个数组的首位：
1. 指针指向元素相同，加入数组，移动两个指针
2. 数组1指向元素小于数组2，数组1指针向后移
3. 数组1指向元素大于数组2，数组2指针向后移

```
class Solution {
    public int[] intersect(int[] nums1, int[] nums2) {
        int n1 = nums1.length;
        int n2 = nums2.length;

        Arrays.sort(nums1);
        Arrays.sort(nums2);
       
        int l1 = 0,l2 = 0;

        List<Integer> list = new ArrayList();
       

        while (l1 < n1 && l2 < n2) {
            
            if(nums1[l1] == nums2[l2]) {
                list.add(nums1[l1]);
                l1++;
                l2++;
            }else if(nums1[l1] < nums2[l2]) {
                l1++;
            }else if(nums1[l1] > nums2[l2]) {
                l2++;
            }

        }

        int[] ans = new int[list.size()];
        for (int i = 0 ; i < list.size() ; i++) {
            ans[i] = list.get(i);
        }
        return ans;
    }
}
```
